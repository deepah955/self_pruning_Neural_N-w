How It Works

Every weight w is paired with a learnable gate score s:
gate = σ(s) ∈ (0, 1) The actual computation uses pruned_weight = w × gate. A sparsity penalty λ · Σ gate is added to the cross-entropy loss. Because the L1 gradient does not vanish as the gate approaches zero, the optimiser keeps pushing unused gates all the way to 0 — permanently silencing that weight.

L1 vs L2 — Why L1 Causes Exact Zeros

Penalty Gradient on score Behaviour at gate→0
L2 (Σ gate²) −2 · gate · σ' Gradient shrinks → rarely hits 0 exactly
L1 (Σ gate) −σ'(score) Constant pull → collapses to 0
