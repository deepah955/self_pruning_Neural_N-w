How It Works

Every weight w is paired with a learnable gate score s: gate = sigmoid(s)   in (0, 1) The forward pass computes pruned_weight = w × gate, then runs the standard linear operation. Adding a sparsity penalty λ • Σ gate to the cross-entropy loss creates constant gradient pressure that drives unused gates exactly to 0 — permanently removing those weights from the network.

L1 vs L2 — Why L1 Causes Exact Zeros

Penalty Gradient on score Behaviour as gate → 0
L2 (Σ gate²) -2 • gate • σ' Gradient shrinks — gate rarely reaches exactly 0
L1 (Σ gate) -σ'(score) Constant pull — gate collapses fully to 0
